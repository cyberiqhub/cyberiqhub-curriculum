# 🌐 CyberIQ Hub – Networking Curriculum

## Course Overview

The Networking course introduces students to the fundamental concepts of computer networking. Students will learn how devices communicate, understand common networking technologies, gain hands-on experience with network tools, and build the foundational knowledge needed for cybersecurity, IT, and CyberPatriot competitions.

---

## Prerequisites

- Basic computer skills
- Curiosity and willingness to learn
- No prior networking experience required

---

## Learning Objectives

By the end of this course, students should be able to:

- Explain how computer networks function
- Understand common networking terminology
- Identify different types of network devices
- Configure basic network settings
- Troubleshoot simple network issues
- Use networking tools for diagnostics
- Build a strong foundation for cybersecurity
